# 皮影戏数字化保存与远程提线演练系统

## 项目简介

本系统是一个面向民间非遗皮影戏的数字化保存与远程提线演练平台，采用前后端分离架构，融合物理引擎、3D渲染和实时通信技术。

## 系统架构

```
┌─────────────────────────────────────────────────────────────┐

│  ┌──────────────┐  ┌──────────────┐  ┌──────────────────┐  │
```

## 核心功能

### 后端模块

1. **皮质动力学物理引擎** (`backend/src/physics_engine.py`)
   - 模拟牛皮材质的弹性、阻尼、质量等物理属性
   - 20个关节的完整皮影骨骼系统
   - 弹簧-阻尼动力学模型
   - 支持脉冲式提线拉力曲线

2. **WebSocket实时服务** (`backend/src/websocket_server.py`)
   - 60fps 关节数据流推送
   - 多客户端独立会话管理
   - 支持7种消息类型
   - 自动重连和心跳检测

### 前端模块

1. **3D皮影渲染器** (`frontend/src/composables/useShadowPuppetRenderer.ts`)
   - Three.js PBR材质渲染
   - 牛皮质感+边缘描边效果
   - 5根提线动态跟随
   - 多光源投影系统

2. **动画平滑驱动**
   - 四元数球面线性插值 (SLERP)
   - 可调节的动画平滑度
   - 关节受力发光反馈

3. **交互系统**
   - 键盘快捷键序列识别
   - 400ms序列超时窗口
   - 最长匹配优先算法

## 快速开始

### 启动后端服务

```bash
cd backend
pip install -r requirements.txt
python main.py
```

服务将在 `ws://localhost:8765` 启动。

### 启动前端服务

```bash
cd frontend
npm install
npm run dev
```

前端将在 `http://localhost:5173` 启动。

### 运行单元测试

```bash
cd backend
python tests/test_physics.py
```

## 物理引擎原理

### 皮质材料参数

- 杨氏模量: 1.5e6 Pa
- 厚度: 3mm
- 阻尼系数: 45.0
- 质量面密度: 0.8 kg/m²

### 动力学方程

```
τ_total = τ_spring + τ_damping + τ_gravity + τ_air + τ_tension
α = τ_total / I
ω += α * dt
θ += ω * dt
```

其中：
- `τ_spring`: 弹性恢复力矩 (胡克定律)
- `τ_damping`: 阻尼力矩 (与角速度成正比)
- `τ_gravity`: 重力力矩
- `τ_air`: 空气阻力力矩 (与角速度平方成正比)
- `τ_tension`: 提线拉力力矩

## 特色亮点

1. **真实物理模拟**: 基于牛皮材质参数的动力学模型，还原皮影的飘动质感
2. **低延迟通信**: 60fps实时数据流，毫秒级延迟
3. **平滑动画**: 四元数SLERP插值，避免万向节锁
4. **组合招式系统**: 支持单键、双键、三键连招，模拟真实提线操作
5. **视觉反馈**: 关节受力时材质发光，直观显示拉力大小
6. **非遗数字化**: 完整的皮影骨骼系统，为传统艺术数字化保存提供技术方案
